Bower
bower install --save-dev embed-js
Manual Installation

  <!--======== css ============-->
  <link rel="stylesheet" href="path/to/embed.css"/>

  <!--===============js==================-->

JavaScript
Creating an instance. 

  var x = new EmbedJS({
    element: document.getElementById('element'),
    googleAuthKey : 'xxxx'
  })
Note : To know about all the available options, go here.
Rendering the result in the same block 

  //Render the result
  x.render();
Getting the resulting string 

  //Get the resulting string
  x.text(function(data){
    console.log(data); //The resultant string
  })
destroying the instance 

  //Destroy the instance
  x.destroy()
