Default Methods

This method of usage should be preferred if you wan't to apply embed.js to a single block or want the result as string.

.render()
Renders the result in the element from which the original raw string was taken. 

  <div id="element">
    Lotus :smile: eleates vix attrahendams luna est.Advenas mori!Fermiums prarere in cubiculum!Cum cacula cantare,
    omnes stellaes manifestum azureus, nobilis https://angularjs.org/ acipenseres.Cum orgia mori, omnes rationees
    :) experientia alter, regius :heart: mortemes.Devatios persuadere, tanquam secundus spatii.Heu,
    barcas!Cedriums observare!A falsis, lacta talis imber. :P Cur eleates peregrinatione?
  </div>

  <script>
  var x = new EmbedJS({
    element: document.getElementById('element'),
    googleAuthKey : 'xxxx'
  })

  x.render()
  </script>

Here the render() method will replace the html content of #element with the processed string.


.text(function(result, input){ })
Syntax : .text(function(result, input){})

This method comes handy if you want to only get the resulting string and use it according to your wish.


  var x = new EmbedJS({
    element: document.getElementById('element'),
    googleAuthKey : 'xxxx'
  })

  x.text(function(result, input){
    console.log(result) //result string
    console.log(input) //input string
  })
.destroy()
The destroy() method turns off all the event listeners and replaces the string with the original string. 

  var x = new EmbedJS({
    element: document.getElementById('element'),
    googleAuthKey : 'xxxx'
  })

  x.render() //renders the result
  x.destroy() //destroys the instance

ejs Methods

The following methods should be used when you want to use the plugin on multiple blocks.

.setOptions(options)
Can be used to set the options globally. The options can be overridden by passing the option for a single block. 

  ejs.setOptions({
    videoDetails:false,
    videoJS:true,
    videoWidth:720,
    videoDetails:true,
    googleAuthKey:'xxx',
    inlineEmbed:[],
    marked:true,
    link:true
  });

  var x = new EmbedJS({
    element: document.getElementById('block')
  });
.applyEmbedJS(className)
Can be used to apply this plugins to multiple classes by passing the class name as string. It will use the globally set options. 

  ejs.setOptions({
    videoDetails:false,
    videoJS:true,
    videoWidth:720,
    videoDetails:true,
    googleAuthKey:'xxx'
  });

  ejs.applyEmbedJS('.element')
.destroyEmbedJS()
destroys all the instances of the plugin 
ejs.destroyEmbedJS()
